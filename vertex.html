VERTEX PLAYER

Visit Github

The Vertex Player started off as a Final Assignment for the Web Design Program at Humber College. The original aim was to create a web-based personal media player that was easy and natural to use. We had aimed to use motion through the Leap Motion but found that the technology was still a little finicky, on top of that a user wouldnâ€™t be able to sit on their couch and control the interface since they need to have the leap motion right in front of them.

After the assignment due date we decided to do a not only a redesign of the UI but a complete redesign on the backend as well. Switching from storing movie data into a JSON file to having it store in a MongoDB allowed us to dynamically load content onto the page, as well as easily sort, edit, and delete the content. In its current state it only has movies, but in the future we aim to expand the video section to include TV Shows, Home Videos, as well as a Music section and a Photo Section.
